#Game Joystick Axis Sensor Module
KY-023

#Schematic
* Arduino GND --> Module pin -
* Arduino +5V --> Module pin +5V
* Arduino Analoog 0 --> Module VRx
* Arduino Analoog 1 --> Module VRy
* Arduino Digital --> Resistor??-->Module SW
